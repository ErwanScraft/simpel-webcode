**Simpel WebCode** adalah editor kode berbasis web yang minimalis namun bertenaga. Aplikasi ini memungkinkan pengguna untuk menulis, menguji, dan melakukan debugging pada kode **HTML, CSS, dan JavaScript** secara instan dalam satu alur kerja yang mulus.

## ğŸŒŸ Fitur Utama

* **Real-time Rendering:** Perubahan kode langsung diproses dan ditampilkan di panel pratinjau tanpa perlu memuat ulang halaman.
* **Powered by CodeMirror 6:** Pengalaman menulis kode yang superior dengan fitur editor modern, *syntax highlighting*, dan manajemen teks yang efisien.
* **Multi-Tab Interface:** Navigasi cepat antara editor HTML, CSS, dan JavaScript.
* **Clean & Responsive:** Antarmuka yang bersih dan fokus pada produktivitas, nyaman digunakan di berbagai perangkat.
